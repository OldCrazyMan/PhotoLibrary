# PhotoLibrary
Searcher photos from api.unsplash.com

## The project uses:
+ [Unsplash.com API](https://api.unsplash.com)
+ JSON & Codable 
+ SDWebImage pods
+ MVC
+ WaterfallLayout
+ Code layout (without storyBoard).

## Installation:

1. Open the console/terminal, go to the project folder (—Åopy the project address in console/terminal):
cd $

2. Copy the project address to the console/terminal and execute the command:
$ pod install
